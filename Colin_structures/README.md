# Colin's 3DNA Fiber Diffraction Structures

These are DNA structures generated by 3DNA's fiber program, organized by form:

- `A_form/xyz/` — A-DNA structures (64 palindromic 6-bp sequences)
- `B_form/xyz/` — B-DNA structures (64 palindromic 6-bp sequences)
- `Z_form/xyz/` — Z-DNA structures (64 palindromic 6-bp sequences)

## XYZ File Format

Each `.xyz` file contains a double-stranded DNA hexamer. The atom ordering is:

### Heavy atoms (non-hydrogen)

1. **Strand I** (5'→3'): All heavy atoms for each nucleotide in sequence order
2. **Strand II** (5'→3' of strand 2 = **reverse complement** of strand I sequence):
   All heavy atoms for each nucleotide

### Per-nucleotide atom ordering (3DNA convention)

Backbone (11 atoms):
```
P, O1P(OP1), O2P(OP2), O5', C5', C4', O4', C3', O3', C2', C1'
```

Base atoms follow (varies by base type):
- **A** (10 atoms): N9, C8, N7, C5, C6, N6, N1, C2, N3, C4 → total 21
- **T** (9 atoms):  N1, C2, O2, N3, C4, O4, C5, C5M, C6 → total 20
- **G** (11 atoms): N9, C8, N7, C5, C6, O6, N1, C2, N2, N3, C4 → total 22
- **C** (8 atoms):  N1, C2, O2, N3, C4, N4, C5, C6 → total 19

### After heavy atoms

3. **Hydrogen atoms** (strand I then strand II)
4. **Terminal O atoms** (2 atoms: 5' terminal oxygens for each strand)
5. **Terminal H atoms** (2 atoms)

## Important Notes

- **Strand 2 ordering**: Strand 2 is stored as the **reverse complement** sequence
  in 5'→3' direction. For example, if strand 1 is 5'-ATATAT-3', strand 2 is
  stored as 5'-ATATAT-3' (reverse complement), NOT as TATATA (complement).
  
- **All sequences are palindromic** (restriction enzyme recognition sites),
  so complement == reverse complement for these files. But the atom ordering
  follows the reverse complement convention.

- The XYZ files have **element symbols only** (P, O, C, N, H) — no atom names.
  Atom names must be inferred from the known ordering above.

- **Charge model**: The terminal phosphate groups have been adjusted so that
  each structure has exactly **−1 formal charge per nucleotide**. The 5' terminal
  phosphate gets an extra oxygen (O5T) to complete the PO₄ group, ensuring
  a systematic −1 charge per nucleotide. This makes the structures suitable
  for computational chemistry setups (e.g., molecular dynamics, QM/MM).
  Total charge = −(number of nucleotides) = −(2 × sequence_length).
