# DNA Builder

A Python tool for generating accurate A-, B-, and Z-form double-stranded DNA structures from a nucleotide sequence.

## Motivation

Existing tools like Avogadro's built-in DNA builder only correctly handle B-form DNA. When asked to build A-DNA or Z-DNA, they typically only change the rise between base pairs without adjusting the many other geometric parameters that differ between DNA forms (sugar pucker, glycosidic angle, base pair inclination, x-displacement, etc.). This produces geometrically incorrect structures.

**DNA Builder** addresses this by using the proper helical parameters and backbone conformations for each DNA form, based on fiber diffraction studies (Arnott & Hukins, 1972; Arnott et al., 1980) and the base pair reference frame approach (Olson et al., 2001; Lu & Olson, 2003).

## Features

- **Three DNA forms**: A-DNA, B-DNA, and Z-DNA with correct geometry
- **Proper backbone conformations**:
  - B-DNA: C2'-endo sugar pucker, anti glycosidic angle
  - A-DNA: C3'-endo sugar pucker, anti glycosidic angle
  - Z-DNA: alternating C3'-endo/syn (purines) and C2'-endo/anti (pyrimidines)
- **Correct helical parameters**:
  - B-DNA: rise = 3.375 Å, twist = 36.0°, 10 bp/turn
  - A-DNA: rise = 2.548 Å, twist = 32.7°, 11 bp/turn, inclination = 19.1°
  - Z-DNA: left-handed helix, dinucleotide repeat, 12 bp/turn
- **Watson-Crick base pairing** with correct C1'-C1' distance (~10.4 Å)
- **Multiple output formats**: PDB, XYZ, CML
- **Batch-friendly**: command-line interface for scripting

## Usage

### Command Line

```bash
# Build B-DNA (default)
python -m dna_builder ATCGATCG --output b_dna.pdb

# Build A-DNA
python -m dna_builder ATCGATCG --form A --output a_dna.pdb

# Build Z-DNA (use alternating purine-pyrimidine sequence)
python -m dna_builder GCGCGCGC --form Z --output z_dna.pdb

# Output in XYZ format
python -m dna_builder ATCGATCG --format xyz --output dna.xyz

# Output in CML format (for Avogadro)
python -m dna_builder ATCGATCG --format cml --output dna.cml

# Print to stdout
python -m dna_builder ATCGATCG
```

### Python API

```python
from dna_builder.builder import build_dna
from dna_builder.io_pdb import write_pdb

# Build B-DNA
atoms = build_dna("ATCGATCG", form="B")
write_pdb(atoms, "b_dna.pdb", title="B-DNA ATCGATCG")

# Build A-DNA
atoms = build_dna("ATCGATCG", form="A")
write_pdb(atoms, "a_dna.pdb")

# Build Z-DNA
atoms = build_dna("GCGCGCGC", form="Z")
write_pdb(atoms, "z_dna.pdb")
```

### Command-Line Options

```
positional arguments:
  sequence              DNA sequence for strand I (5' to 3'). Only A, T, G, C.

optional arguments:
  --form {A,B,Z}, -f    DNA form (default: B)
  --output FILE, -o     Output filename (default: stdout)
  --format {pdb,xyz,cml}, -F
                        Output format (default: pdb)
```

## How It Works

### Construction Algorithm

1. **Base pair construction**: Each Watson-Crick base pair is built in a reference frame with the origin at the base pair center. Standard base geometries from Clowney et al. (1996) are used. The complementary strand is generated by the pseudo-dyad operation: (x, y, z) → (-x, -y, -z).

2. **Backbone attachment**: Sugar-phosphate backbone atoms are attached to each base using templates specific to each DNA form:
   - C2'-endo sugar pucker for B-DNA and Z-DNA pyrimidines
   - C3'-endo sugar pucker for A-DNA and Z-DNA purines
   - Anti glycosidic angle for A/B-DNA and Z-DNA pyrimidines
   - Syn glycosidic angle for Z-DNA purines

3. **Helical positioning**: Each base pair is positioned relative to the helix axis by applying x-displacement and inclination, then the helical screw operation (rotation by twist + translation by rise along Z) generates successive base pairs.

4. **Z-DNA specifics**: Z-DNA uses a dinucleotide repeat with alternating rise/twist values (-9°/3.53 Å and -51°/3.89 Å), summing to -60°/7.42 Å per dinucleotide.

### Key Geometric Parameters (Validated)

| Parameter | B-DNA | A-DNA | Z-DNA |
|-----------|-------|-------|-------|
| Rise (Å) | 3.375 | 2.548 | 3.53/3.89 |
| Twist (°) | 36.0 | 32.7 | -9/-51 |
| bp/turn | 10 | 11 | 12 |
| Handedness | Right | Right | Left |
| C1'-C1' (Å) | 10.35 | 10.35 | 10.40 |
| Inclination (°) | -5.9 | 19.1 | -6.2 |
| Sugar pucker | C2'-endo | C3'-endo | Alternating |
